# Block styles

We want most of our blocks of information to look the same.
Instead of creating a block component, we're simply sharing some styles that can be imported and reused.

## Features

* `:host` with `display:grid`, border and radius
* `.cc-block_title` with color, font-size, font-weight and some margin.

## How to use it?

In your component, import the styles like this:

```js
import { blockStyles } from 'components/styles/block.js';
```

Then you need to use it in your LitElement `static get styles()` like this:

```js
static get styles () {
  return [
    blockStyles,
    css`
      /* CSS for component goes here */
    `  
  ]
}
```

With this setup, the `:host` will be styled automatically.
You'll have to use the `.cc-block_title` CSS class on your title and the other children of your host will be placed in the grid with the right gap between them.
